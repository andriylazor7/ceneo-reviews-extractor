The Ceneo Review Extractor is a web application designed to extract product reviews from the Ceneo platform, analyze the data, and present valuable insights. It provides an intuitive user interface to easily scrape reviews, visualize key data points, and download review data in multiple formats.

Purpose of the Application

The purpose of this application is to help users gather valuable product reviews from Ceneo, analyze trends, and make more informed purchasing decisions based on detailed reviews. By extracting opinions, advantages, and disadvantages, this tool makes review analysis easier and more insightful.

How the Application Works

This application allows users to input a product ID from Ceneo, which triggers the review extraction process. The scraper collects reviews, categorizes opinions, and calculates averages like the overall rating, advantages, and disadvantages of the product. After extraction, users can view the reviews, generate charts, and download the data in CSV, JSON, or Excel formats.

Libraries Used
  • BeautifulSoup - For web scraping and extracting review data.
  • Requests - For making HTTP requests to the Ceneo pages.
  • Flask - For building the web application and handling routing.
  • Matplotlib - For generating visual charts, such as pie charts and bar graphs.
  • pandas - For handling, cleaning, and storing data in CSV, JSON, and Excel formats.
  • openpyxl - For working with Excel files (.xlsx).
Applied Approaches

The application uses an object-oriented approach for modular code management, with classes like `Scraper`, `Product`, `Opinion` and `Visualizer` to organize different aspects of the review collection and analysis. Pagination handling is implemented to ensure that all product reviews are collected from multiple pages, however Ceneo.pl has a block that prevents extraction of all reviews, while data validation and duplicate filtering ensure accuracy. Additionally, Flask is used to create a dynamic web interface, allowing users to interact with the application through buttons, forms, and downloadable files.
